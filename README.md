# Chatbot Smith Website

Static HTML website for Chatbot Smith — a custom AI chatbot service for small businesses.

## File structure
- index.html — Homepage
- pricing.html — Full pricing page with tier cards and FAQ
- demo.html — Live demo page with Studio Elara salon demo
- how-it-works.html — Process breakdown and FAQ accordion
- book-a-call.html — Contact form / booking page
- about.html — About page with founder story and values
- css/styles.css — All shared styles (colors, fonts, layout)
- js/nav.js — Navigation component (edit nav here — updates all pages)
- js/footer.js — Footer component (edit footer here — updates all pages)
- js/main.js — Tab switching, FAQ accordion, form handler
- demos/salon/index.html — Studio Elara fictional salon demo site

## How to open locally
Double-click index.html — no server needed.
